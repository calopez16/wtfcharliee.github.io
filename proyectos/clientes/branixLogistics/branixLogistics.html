BnxLogistics

Información del proyecto:

Sistema web desarrollado para control interno de procesos para la cualquier empresa o cliente que desee gestionar y controlar el transporte de mercancias y manejo de unidades, además de contar con módulos que permiten administrar diferentes áreas como inventarios, empleados, equipos, nómina, facturación, etc.

Detalles del proyecto:

  • Empresa:Branix.
  • Cliente:Cualquier cliente en el área transportista.
  • Tecnologías:Asp .NET Core 3.1 + C#, EntityFrameWork, HTML5, CSS3, JavaScript, SignalR.
  • Base de datos:SQL Server
  • Fecha Inicio:Enero, 2022
