Sistema de administración HE

Información del proyecto:

Sistema web desarrollado para control interno de proceso de manofactura dentro de la empresa IQS ademas para gestión de empleados, generación de reportes, control de usuarios por permisos, etc.

Detalles del proyecto:

  • Cliente:IQS Services
  • Tecnologías:Asp .NET Core 3.1 + C#, EntityFrameWork, estructura de repositorio, HTML5, CSS3, JavaScript.
  • Base de datos:SQL Server
  • Fecha:Abril, 2021
