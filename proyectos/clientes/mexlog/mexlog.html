Mexlog 3.0

Información del proyecto:

Sistema web desarrollado para control interno de procesos para la empresa Mexicana Logistics el cual permite gestionar diferentes áreas como monitoreo de transporte de unidades, inventarios, empleados, equipos, nómina, facturación, etc.

Detalles del proyecto:

  • Empresa:Branix.
  • Cliente:Mexicana logistics.
  • Tecnologías:Asp .NET Core 3.1 + C#, EntityFrameWork, HTML5, CSS3, JavaScript, Firebase.
  • Base de datos:SQL Server
  • Fecha Inicio:Marzo, 2020
  • Fecha de implementación:Julio, 2021

Comentarios:

En este proyecto fui uno de los programadores principales para darle función y diseño a la parte visual del sistema, pionero en la estandarización de las vistas e incluso fragmentos de código los cuales nos permitian poder desarrollar módulos con mayor facilidad.
