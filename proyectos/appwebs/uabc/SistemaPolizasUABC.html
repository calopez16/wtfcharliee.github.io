Página Hielo Estrella

Información del proyecto:

Sistema web para el control y digitalización de pólizas digitalizadas, nos permite organizar las polizas por caja e imprimir etiquetas para su organización física. Además de tener un módulo para firmar pólizas por medio de un certificado electrónico.

Detalles del proyecto:

  • Cliente:Universidad Autónoma de Baja California(UABC)
  • Tecnologías:MVC EntityFramework + C#, HTML5, CSS3 y JavaScript
  • Fecha:Septiembre, 2019
